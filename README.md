# Clinic Management System

## Overview
This is a web-based Clinic Management System using HTML, CSS, JS, and Firebase. It allows doctors and receptionists to manage patient tokens, information, prescriptions, and billing.

## Technologies
- HTML, CSS, JS
- Firebase (Auth, Firestore)

## Workflow
1. User logs in as doctor or receptionist.
2. Receptionist adds patients and generates tokens.
3. Doctor views patients and adds prescriptions.
4. Billing generates bills based on patient data.

## Execution
1. Clone the repo.
2. Open `index.html` in a browser.
3. Set up Firebase config in `js/firebase-config.js`.
4. Deploy to Firebase Hosting: `firebase deploy`.
